# @idle-engine/content-test-game

Internal stress-test content pack for exercising `@idle-engine/core` features in a playable shell scenario.

## Play

From the repo root:

```bash
pnpm test-game:play
```

## Content authoring

- Source content lives in `packages/content-test-game/content/pack.json`.
- Generated artifacts are refreshed via `pnpm generate` from the repo root.
